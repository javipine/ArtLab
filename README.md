# ArtLab
Web de Diseño Interactivo de Arte para el Diseño Grafico de Estampados de Moda en prendas de vestir
# 👕 Interactive Fashion Collection 2026

![Licencia](https://img.shields.io/badge/license-MIT-blue.svg)
![Estado](https://img.shields.io/badge/status-en%20desarrollo-green.svg)

Una experiencia interactiva de moda que permite a los usuarios visualizar y personalizar estampados sobre prendas en tiempo real. Este proyecto combina diseño textil con tecnologías web modernas para eliminar la barrera entre el concepto y el producto final.

## ✨ Características Principales

* **Drag & Drop Interactivo:** Arrastra cualquier diseño (.png, .jpg) directamente sobre la prenda.
* **Modo de Mezcla Realista:** Los estampados se adaptan a las sombras y arrugas de la tela usando `mix-blend-mode`.
* **Snapshot Engine:** Generación de vista previa descargable para redes sociales o producción.
* **Interfaz Minimalista:** Diseño enfocado en la prenda, optimizado para una experiencia de usuario (UX) fluida.

## 🚀 Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura semántica y diseño responsivo.
* **JavaScript (Vanilla):** Lógica de interacción y manejo de archivos en el cliente.
* **html2canvas:** Biblioteca para la captura y exportación de imágenes.
* **GitHub Pages:** Despliegue automático de la aplicación.
